CPU emulator.

Commands summary (V1.12):
: {value} - sets label {value}
PROC : {value} - sets pointer to function {value}
PUSH {value}- pushes value to stack
POP (reg) - pops value out of stack (or to reg)
IN - gets value from the input stream and pushes it into stack
OUT - prints popped value on screen
DUP - duplicates top value of stack

ADD - pushes sum of 2 popped elements
SUB - pushes difference of 2 popped elements
MUL - pushes product of 2 popped elements
DIV - pushes quotient of 2 popped elements
POW - pushes power function of 2 popped elements
SIN - pushes sin of last element instead of it
COS - pushes cos of last element instead of it
SQRT - pushes sqrt of last element instead of it

AND - logical and
OR - logical or
NOT - logical not

EQU - ==
LT - <
GT - >

JMP :{value} - jumps on {value} label
JE :{value} - jumps on {value} label if pop = pop
JNE :{value} - jumps on {value} label if pop != pop
JA :{value} - jumps on {value} label if pop > pop
JAE :{value} - jumps on {value} label if pop >= pop
JB :{value} - jumps on {value} label if pop < pop
JBE :{value} - jumps on {value} label if pop <= pop

CALL :{value} - calls {value} function
RET - returns from current function

SPC - prints ' '
TAB - prints ''
EOL - prints '
'

DUMP - prints full dump info on CPU

END - ends the program
