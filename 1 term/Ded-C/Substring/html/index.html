Substring search

The task.
The program gets 2 strings and demonstrates some functions that are identical to standart string.h functions
Realised functions:
  • get_str() gets a string from the input
  • str_cmp() compares 2 strings
  • sub_str() searches a substring
  • str_len() returns length of the string Return values are documented and usually are equal to standart ones from string.h
