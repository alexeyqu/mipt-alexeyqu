Задание 1: Поиск максимума и минимума радиус векторов в комплексной матрице

 Требуется:
1. Написать программу на языке C с использованием POSIX Threads, которая будет находить и распечатывать значения в матрице с максимальным и минимальным модулем радиус вектора, и распечатывать позиции, в которых данные значения найдены (предполагается, что программа будет написана так, чтобы делать это за кратчайшее время).
2. Написать программу на языке C без использования каких-либо механизмов распараллеливания кода, которая делает то же, что и программа из предыдущего пункта. 
3. Построить график зависимости времени работы программы (без учёта времени на считывание матрицы из файла)  от  размерности матрицы (предполагаем, что матрица квадратная) и числа задействованных потоков. 
4. Написать генератор матриц, для проверки работоспособности программы и построения графика.

Матрица подается на стандартный поток ввода программ в виде текстового представления чисел, разделяемых пробельными символами, первое число — размерность квадратной матрицы (должно быть натуральным), все последующие — вещественные, каждая пара чисел обозначает элемент матрицы (первое число — действительная часть, второе — мнимая). Подавать на вход матрицу программе можно через пайпы коммандной оболочки (shell'а).

К сдаваемому заданию приложить описание аппаратной архитектуры, на которой проводилось тестирование, а так же табличку для диапазона по числу потоков {1, 2, 3, 4, 6, 8, 16, 24, 32} и  по  размерности матрицы {3,10,100,500,1000,3000,5000}. К табличке добавить поле  «sec» - где указывается время работы программы из пункта 2 на том же наборе диапазонов размерности матрицы. 

При написании программ рекомендуется придерживаться одного из нижеперечисленных стилей программирования:

http://www.openbsd.org/cgi-bin/man.cgi/OpenBSD-current/man9/style.9?query=style&arch=i386
https://www.kernel.org/doc/Documentation/CodingStyle

При проверке заданий будет учитываться плагиат: схожие работы не будут засчитываться.
